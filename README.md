## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- npm/yarn
- Modern browser with WebGL support

### Installation

```bash
git clone https://github.com/yourusername/trigonometric-art-studio.git
cd trigonometric-art-studio
npm install
npm run dev
```
